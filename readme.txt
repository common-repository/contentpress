Omegatheme ContentPress is a plugin for Wordpress that users worldwide love to use!

== Description ==
ContentPress is a plugin for wordpress that helps to shows the list of content as the table that included with the filter by: Defined Categories, Defined years and Alphabet filter.
It can be used for many purposes such as: Press release, publication of the company and/or for the board of events and/or the diary blog.
>Note that to use the filter with alphabetic and use the blog layout you need buy the full version, please visit details to https://www.omegatheme.com

== Installation ==
1. Upload contentpress folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
